Optimizing KWS CNN Architecture for ESP32 and Microcontroller Units

M. A. Abdelraouf
Mohamed AI Laboratory Research Journal — Published June 2026

This paper outlines a structured design process for optimizing Mel-Frequency Cepstral Coefficients (MFCC) feature extraction and 2D Convolutional Neural Networks for voice keyword spotting on low-power devices. Evaluates INT8 post-training quantization on RAM/ROM bounds, achieving 98% detection on target wake-words under 100KB heap constraints.

Distributed SWARM Inference: Offloading LLM Workloads to Client-Side Swarms

M. A. Abdelraouf
Lab Technical Report #24 — Shipped Code Integration

Investigates a decentralized browser-bridged computing topology. Describes using WebSockets to segment LLM prompt parameters, distributing token processing weights to client swarms consisting of multiple local machines (Swarm Groups) to bypass expensive commercial clouds.

EMG Signal Extraction & Amplitude Envelope Tracking for Prosthetic Control

M. A. Abdelraouf
Exhibited at Pharos University Biomedical Research Day

Focuses on biological signal acquisition and filtering circuits. Proposes an embedded C envelope extraction algorithm running on Atmega microcontrollers to process differential surface electrode potentials, generating robust muscle thresholds for servo mechanical hands.
